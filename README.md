# TrueMark-main

This is the main repo for TrueMark Suite, an application suite for Optical Mark Recognition (OMR).

This includes:
- TrueMark Desktop [Changelog](desktop_app/desktop_app_changelog.md)
- TrueMark Android
- TrueMarkDb (for the future) 

## For Developers
!NOTE!
For python scripts, create a virtual environment with this command:

```bash
python3 -m venv path/to/venv/
```
Dependencies:
- pandas
- pillow (for barcode generation)

Install all dependencies with pip3.
```bash
pip3 install pandas
pip3 install python-barcode 
pip3 install pillow
```
